## アイデア1: 「和モダン・禅庭」アプローチ

### Design Movement
日本庭園の「侘び寂び」と現代ミニマリズムの融合。枯山水の静謐さをデジタルで表現。

### Core Principles
1. **余白の美学**: 情報を詰め込まず、呼吸できる空間を確保
2. **自然素材の質感**: 木目、石、苔のテクスチャを控えめに活用
3. **非対称バランス**: 完璧な左右対称を避け、動きのある構図
4. **一期一会の緊張感**: CTAに向かう導線を「道」として設計

### Color Philosophy
- **主調**: 深い墨色（#1a1a1a）と生成り白（#f5f3ef）
- **差し色**: 苔の緑（#4a6741）- 生命力と再生を象徴
- **アクセント**: 朱色（#c53d43）- CTAに使用、神社の鳥居のような導きの色
- **感情意図**: 落ち着き、信頼、職人の誠実さ

### Layout Paradigm
- 縦スクロールを「庭を歩く」体験として設計
- セクション間に「飛び石」的な視覚的休憩点
- 非対称グリッド：テキスト60% / ビジュアル40%の黄金比

### Signature Elements
1. **墨流しグラデーション**: セクション背景に和紙のような淡いテクスチャ
2. **円窓フレーム**: 重要な情報を円形の窓から覗くように配置
3. **筆文字風見出し**: 手書き感のあるフォントで温かみを演出

### Interaction Philosophy
- ホバー時は「水面の波紋」のような静かな広がり
- スクロールは「庭を歩く」ようにゆったりと
- タップは「石を踏む」確かな手応え

### Animation
- セクション出現：opacity + translateY（下から浮き上がる）、0.6秒、ease-out-cubic
- カード：影が深くなる + 微細なscale（1.01）
- 数字カウントアップ：筆で書くような左から右への出現
- prefers-reduced-motion対応で簡素化

### Typography System
- 見出し：Noto Serif JP（明朝体）- 格式と伝統
- 本文：Noto Sans JP（ゴシック）- 可読性優先
- 数字：Outfit（英字）- モダンさのアクセント
- 階層：h1=2.5rem, h2=1.75rem, h3=1.25rem, body=1rem

## 選定理由

**アイデア3「プロフェッショナル・クラフト」を採用**

### 採用理由
1. **ターゲット適合性**: 40〜70代の戸建て持ち家層は「信頼できる職人」を求めている。過度な装飾より、実直さと専門性が響く
2. **CVR最適化**: 明確な情報階層、目立つCTA、比較表などがコンバージョンに直結
3. **軽量化との両立**: シンプルな構造でありながら、斜めカットやバッジで視覚的な差別化が可能
4. **地域密着感**: 冷たいコーポレートデザインではなく、「地元の頼れる職人」感を演出できる

### 実装上の強み
- CSS変数でカラーパレットを一元管理
- clip-pathとtransformのみでダイナミックな演出
- システムフォント + Noto Sans JPで軽量かつ日本語最適化
- IntersectionObserverで効率的なスクロールアニメーション
